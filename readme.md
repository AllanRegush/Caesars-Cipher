# Caesar Cipher 

[Inspired by Odin Project Assignment] (https://www.theodinproject.com/courses/ruby-programming/lessons/building-blocks#project-1-caesar-cipher)

### Dependencies

1. [Google Test] (https://github.com/google/googletest)

### To build

``` bash
mkdir build
cd build
cmake ..
```

### To Test
``` bash
make all
ctest
```

### To Run
``` bash
cd build
./caesar
```
